This directory contains a copy of version 5.1.0 of the jQuery Datepicker package
(see http://keith-wood.name/datepick.html). It is released by the author under
the MIT licence.

The ZIP file is included in this repository and its contents are intended to
be extracted into the directory containing the ZIP file itself. All other files
within this directory in the repository are ones that have either been modified
from or added to the original Datepicker package.

A copy of the relevant jQuery library script is also contained within the
Datepicker directory. This is also covered by the MIT licence.
